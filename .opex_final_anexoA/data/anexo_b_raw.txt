B00.1. MARCO CONCEPTUAL BIM Estándares y Nomenclatura
El presente anexo adopta la metodología Building Information Modeling (BIM) según:
Estándares normativos:
ISO 19650-1:2018 - Organization and digitization of information (gestión de información)
ISO 19650-2:2018 - Delivery phase of assets (fase de entrega)
ISO 16739-1:2018 - Industry Foundation Classes (IFC) - Esquema de datos
ISO 12006-3:2022 - Building construction - Framework for object-oriented information
PAS 1192-2:2013 - Capital/delivery phase (Reino Unido)
Nomenclatura y serialización de datos:
La estructuración de componentes sigue el estándar BIM-JSON (Building Information Model - JavaScript Object Notation),compatible con:
IFC-JSON: Serialización JSON de Industry Foundation Classes
COBie: Construction Operations Building information exchange
BCF: BIM Collaboration Format (ISO 19650-3)
Referencia técnica: buildingSMART International Data Dictionary (bSDD)
B10.2 Jerarquía BIM para CALE.n_1
La jerarquía de componentes para esta categoría se estructura en 6 niveles:
L5: Red Nacional SNCALE
└── L4: Instancias Municipales

Valor Parcial = 1633 m² ×

85.000/m

SUBTOTAL = Σ (Todos los Valores Parciales de la tabla)

Trazabilidad de Costos:
L0.IC_001 → APU_IC_001 (Anexo C) → Valor Unitario → Valor Parcial → SUBTOTAL L1 → TOTAL L2 → TOTAL L3 → ... → TOTAL L5
