📝 Instrucciones de Uso

  1. Asegúrate de que el servidor local esté activo
  2. Para iniciar el servidor, ejecuta: iniciar_servidor.bat
  3. O desde CMD: python -m http.server 8085
  4. Haz clic en cualquiera de los enlaces arriba
  5. Para detener el servidor: Ctrl + C en la terminal
